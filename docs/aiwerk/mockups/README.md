# AIWerk Customer UI mockups

This directory holds early HTML mockups for the Hermes-based AIWerk Customer UI, short AIWerk CUI.

Customer-facing AIWerk CUI text is German by default because first deployments target German-speaking Swiss SME users.

Boundary:
- This belongs to the AIWerk Hermes runtime/UI track, not the legacy Claude Code `agent-dashboard` workspace.
- Mockups here are reference/design artifacts, not production UI code.
- The intended promotion path is: local Hermes prototype → verified AIWerk/hermes-agent runtime/UI capability → AIWerk/base-agent template/runtime lock and verifier → VPS-hosted customer agents.

Current mockup:
- `hermes-agent-dashboard.html` — Rocky / AIWerk Customer UI concept with chat, session controls, agent controls, history, file attachment, and approval card.
